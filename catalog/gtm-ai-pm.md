# 🚀 GTM & AI-PM Prototypes

> The most **under-built** theme relative to demand. AI-native GTM Engineer and AI Product Engineer roles are exploding, and almost no one shows a portfolio project aimed at them. Build one and you're differentiated by default.

**Skills legend:** `GTM automation` · `enrichment + lead scoring` · `persistent memory` · `product prototyping` · `CRM / API integrations` · `LLM-personalized outbound`
**Difficulty:** 🟢 Beginner · 🟡 Intermediate · 🔴 Advanced
**Link types:** 📄 paper · 📘 docs · 🎬 video · 🧑‍🏫 course · 🛠️ tool · 💻 repo

> [!TIP]
> Aiming at these roles specifically? Cross-reference the [GTM Engineer](https://github.com/landedjobs/become-a-gtm-engineer) and [AI Product Engineer](https://github.com/landedjobs/ai-product-engineer-roadmap) roadmaps.

### 1. Context-aware sales agent (voice / email / SMS) — 🟡 Intermediate
An outbound/inbound sales agent with product knowledge and conversation memory across channels. Directly maps to GTM-engineer work.
- **Skills proven:** conversational memory, multi-channel, product-grounded prompting.
- 💻 [filip-michalsky/SalesGPT](https://github.com/filip-michalsky/SalesGPT) — 2.7k ⭐ · MIT · sales-conversation agent.

### 2. Lead-scoring + enrichment agent — 🟡 Intermediate
Given a list of companies/contacts, enrich from public sources and score fit against an ICP with a transparent rubric. The core GTM-engineering loop.
- **Skills proven:** enrichment pipelines, ICP scoring, API orchestration, dedup.
- 💻 [Shubhamsaboo/awesome-llm-apps](https://github.com/Shubhamsaboo/awesome-llm-apps) — 116k ⭐ · Apache-2.0 · enrichment/agent examples.

### 3. Marketing-asset generator (brief → landing page) — 🟡 Intermediate
Turn a one-line brief into copy, a hero image, and a deployable landing page. Shows product + generative + full-stack in one artifact.
- **Skills proven:** structured generation, image gen, code gen, deployment.
- 💻 [crewAIInc/crewAI](https://github.com/crewAIInc/crewAI) — 54.6k ⭐ · MIT · multi-agent content crews.

### 4. Behavior-brief → interactive prototype tool (AI-PM) — 🔴 Advanced
Turn a PM's feature brief into a clickable prototype (spec → components → preview). The "PM who ships" project for AI Product Engineer roles.
- **Skills proven:** spec parsing, code generation, component synthesis, preview.
- 💻 [Shubhamsaboo/awesome-llm-apps](https://github.com/Shubhamsaboo/awesome-llm-apps) — 116k ⭐ · Apache-2.0 · builder/prototype apps.

### 5. Support-ticket classifier with auto-escalation — 🟢 Beginner
Classify inbound tickets by intent/urgency, draft a reply, and escalate the hard ones with a confidence gate. High product value, low build cost.
- **Skills proven:** classification, drafting, confidence-gated routing, integrations.
- 💻 [Shubhamsaboo/awesome-llm-apps](https://github.com/Shubhamsaboo/awesome-llm-apps) — 116k ⭐ · Apache-2.0 · support-agent examples.

### 6. Meeting-notes → CRM-updater agent — 🟡 Intermediate
Transcribe a call, extract action items and deal fields, and push structured updates to a CRM. Voice + extraction + integration — a complete GTM tool.
- **Skills proven:** transcription, extraction, CRM API writes, idempotency.
- 💻 [openai/whisper](https://github.com/openai/whisper) — 104k ⭐ · MIT · transcription front-end.

### 7. Competitor / pricing intelligence monitor — 🟡 Intermediate
Watch competitor pages, detect meaningful changes (filter noise), and deliver a summarized alert. A GTM/PM tool teams actually want.
- **Skills proven:** change detection, noise filtering, summarization, alerting.
- 💻 [Shubhamsaboo/awesome-llm-apps](https://github.com/Shubhamsaboo/awesome-llm-apps) — 116k ⭐ · Apache-2.0 · monitoring agents.
